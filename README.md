# Workshop JavaFX JDBC

# Sobre o projeto
Este projeto é uma aplicação JavaFX, onde desenvolvo um CRUD de Funcionários e Departamentos. 

## Além do CRUD, implemento outras funcionalidades como:
- Modelo de design de aplicativo MVC.
- Padrão DAO.
- Princípios POO.
